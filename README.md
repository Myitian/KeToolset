# KeToolset

## KeSpider

A tool for downloading posts from kemono-like sites.

## FFmpegCheck

A tool for validating video files by fully decoding the entire video.

## MegaDownloaderTaskReset

A tool for repairing corrupted files by resetting the download status in [MegaDownloader](https://megadownloaderapp.blogspot.com/) to re-download the files.

It now includes more QoL features, not just resetting downloads!

Note: The user interface language of this tool is Simplified Chinese.

## MegaDownloaderXmlGenerator

A tool for generating the `DownloadList.xml` file for [MegaDownloader](https://megadownloaderapp.blogspot.com/).
